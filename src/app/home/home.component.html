Simple Caculator

This is a calculator app that uses HTML, CSS and JS. It allows you to do the 4 basic operations with +/- integers or decimals. You can also backspace if you make a typo and clear the calculator.

My 2 Cents Banking/Investing

As a team of about 20 members, we added features and functionalities to this existing banking application. We added functionalities that allow users to help budget their money and invest in crypto/stocks. The project is written in C# for the backend web api and the front end uses Angular.

Mars Rover Photos (using one of NASAs APIs)

This is an app I made using one of NASAs Web APIs (api.nasa.gov). The app uses the API to grab photos from Mars (hidden with show/hide image) on a given sol day and it also shows the ID, Earth Date, Camera and Rover used.
