Linked List Queue in C

About

This is a project I made back in college when learning about data structures. It is a simple queue for customers using a linked list. The project allows you to add customers and then print the list to see the customers in the queue. I guess I can also say I'm pretty efficient with Paint, since I made this handy little image to the left using it.

Coding Language

This program was written in C. I have a header file that defines my two functions. Those functions are to add the customers and to print the customer list. Since C is a very low level programming language, it means that I needed to make use of malloc in order to allocate memory for the program. Finally, in order to point to the next node in the list, I had to use pointers for the nodes in my data structure. The pointers allow us to point to the head, the next nodes, and the tail node to keep track of where every customer is in the queue.
