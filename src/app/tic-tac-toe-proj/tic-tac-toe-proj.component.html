Tic Tac Toe AI Tournament

About

This project takes 15 agents in a tournament setting. The agents have different levels of intelligence. The different levels include: next square (which chooses their move based on the next available square from top-left->bottom-right), blind random (which chooses a random square and goes there if available, if not it will continue until it finds an available square), probability based agents which have a % chance of making an optimal move or it will make either a random move or the next possible square. The final AI is a Minimax agent (Agent 0). This agent is the only agent with the Minimax intelligence, so it should always be the winner of the bracket because it always makes optimal moves. Watch the video to learn more as I go through the project in more depth and show a demo at the end.

Technologies

This app was made using HTML, CSS, and JavaScript. The original project was from a tutorial on youtube where it showcased a Tic-Tac-Toe game you could play with 1 player vs AI. I took out the code for the player and wrote my own code for some of the different AIs.
