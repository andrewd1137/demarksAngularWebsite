Mini Web Pack

About

This is a project I made back in college. It is a simple version of a real webpack tool. Web pack tools allow us to reduce the file size of programs by removing uncecessary information from the file. This includes extra whitespace, uncecessary new lines, comments and combining files that can be combined together so you do not have multiple files. My program removes uncecessary white space, new lines and combines two files that the user selects together.

Coding Language

This program is written in C. It uses character arrays called buffers with size of 1024. You enter two file names and it will parse through the file and remove all uncecessary white space from the code. This is because most new lines and white space in coding is used for us programmers to understand the code. It will also combine the two files into one. This signifcantly reduces the size of larger programs and it is what almost every web service uses in order to make their final projects less size and more efficeient.
