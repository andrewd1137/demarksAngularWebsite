About

This is a calculator app that allows you to do the 4 basic operations with +/- integers or decimals. You can also backspace if you make a typo and clear the calculator. The app has some security features to make sure it runs as intended. Some of these include not being able to put multiple 0s infront of a number unless it's a decimal and allowing users to keep pressing operation buttons with numbers to do the operation without needing to hit ='s every time (Let's you do 2+2+2+2= and will show 8).

Technologies

This app was made using HTML, CSS, and JavaScript. I used the HTMLScriptElement that Angular has to add the scipt file to this HTML Doc. The HTML and CSS from the project were just copied and pasted onto this page in order to display them.
