Recursive GCD Calculator

About

This is a simple project that will give you the Greatest Common Divisor between two numbers (x and y). The project uses recursion to solve this. The way it does this is by calling the function again after it completes. It will keep doing this until it finds the GCD.

Language

This program is written in javascript. Since this project is just a plain javascript file with no HTML or CSS, you will need a runtime environment in order to run the code. This is because javascript needs a runtime environment in order to execute the code writen. Typically, your browser provides this runtime environment, but in this case you can use Node to run the code without use of a browser. It makes the most sense to do this because this is a standalone js file that just uses the console and doesn't have any front end with it.
