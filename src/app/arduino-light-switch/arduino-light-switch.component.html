Arduino Light Switch

About

This is a project I made for my Computer Organization/Computer Architecture Lab. This project is a very simple micro-controller project that just uses a couple of hardware components. It uses an RGB LED, a few wires, a button, a resistor, a shock sensor, and a bread board. I then wrote the code for the button to change colors based on the button click. The shock sensor detects if you hit/shake the snesor and it acts as a 'kill switch' to shutoff the led. It will reset the led back to the beginning sequence. The sequence goes as follows: first button press = static red led, second button press = flashing green light, third button press = static green led, and if at any point you hit the shock sensor, the light will turn off and the next button press will always start back to the static red.

Arduino Coding Language

The Arduino language uses something called INO. It stands for 'Arduino Sketch.' There is virtually no difference between a .INO file and a .CPP (c++) file. There isn't any difference between the programming language or file format, but you do use the Arduino IDE in order to build and run the Arduino projects.
