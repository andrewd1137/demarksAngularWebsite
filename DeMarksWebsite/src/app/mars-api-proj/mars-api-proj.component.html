Mars Rover Photos (using one of NASAs APIs)

About

This is an Angular app I made using one of NASAs Web APIs (api.nasa.gov). The app uses the API to grab photos from Mars on a given sol day and it also shows the ID, Earth Date, Camera and Rover used. There are two ways to view the page and they are displayed below. The first makes use of Bootstrap tables and just displayed the information without images. The second uses Bootstrap cards and displayes the same information, but it also displays the images. You can toggle between the two using the Show/Hide image button (text changes based on which setting you are currently on).

Search a SOL or Get Random SOL

You can find pictures for a SOL day by entering a number into the search bar and clicking the button. The app will then get the pictures from NASAs API and display them onto the screen. You can also get the images by hitting the Random button. This will grab you a random sol day using Math.Random. *Note: Some SOLs do not contain API information so the site will just display nothing if you search a day without pictures. I did not add any error checking for this at the time of this website's release.*

Show/Hide Image

The site will default show you the table in the image above. However, if you would like to see the images (aka the main point of this app), you can click the button to show the images.
